The WebAssembly Working Group invites implementations of three Candidate Recommendations published today:

WebAssembly Core Specification describes version 1.0 of the core WebAssembl standard, a safe, portable, low-level code format designed for efficient execution and compact representation. WebAssembly JavaScript Interface provides an explicit JavaScript API for interacting with WebAssembly. WebAssembly Web API describes the integration of WebAssembly with the broader web platform.
